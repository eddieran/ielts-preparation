# Task 1 Report Templates + Language Bank

Two parts: (1) a structural skeleton for each visual type, and (2) an exhaustive, function-organized language bank (trends, comparison, proportion, approximation, process, maps) with the tense rules.

> **Task 1 basics.** ≥150 words (aim **150–190**), ~**20 min**, worth **one-third** of Writing — so do **Task 2 first**. It is **purely descriptive: no opinion, no conclusion, no reasons.** You *report and compare* what you see. The **overview is what gates Band 7** — an essay with no real overview typically falls to Band 4–5. As with Task 2, these are **structural skeletons, not phrases to memorize** (canned language is disregarded).

---

# Part 1 — The universal structure (all visual types)

4 paragraphs:

1. **Introduction (1 sentence)** — paraphrase the prompt. Swap the given verb ("shows") for *illustrates / depicts / compares / gives information about*, and reword the categories/time frame.
2. **Overview (the Band-7 gate)** — start with **"Overall,"** and give **2–3 big-picture features** (highest/lowest, overall trend, biggest change, overall transformation, number of stages). **NO specific numbers here.**
3. **Body 1** — a logically *grouped* set of data with **selected** figures + comparison.
4. **Body 2** — the remaining data with figures + comparison.

**Golden rules:** *select and group* the significant features (don't list every number); always **compare**, don't just describe; pick the **right tense** (see the cheat sheet at the end).

# Part 2 — Skeleton per visual type

## 1. Line graph (change over time)

Overview = overall direction (up/down), the highest/lowest overall, the biggest change.

```
Intro:     The line graph illustrates [what] [between/from YEAR to YEAR],
           measured in [units].
Overview:  Overall, [the biggest overall trend, e.g. all figures rose].
           In addition, [highest vs lowest category overall].
Body 1:    [Group A]: In [year], [X] stood at [figure], before [rising/falling]
           [adverb] to [figure] by [year]. [Comparison with another line].
Body 2:    [Group B]: In contrast, [Y] [trend + figures]. [Comparison].
```

## 2. Bar chart

Two cases — **read the axis**: if it's over time, treat it like a line graph (trends). If it compares categories at one point, use **comparison** language. Overview = highest/lowest category, biggest gap.

```
Intro:     The bar chart compares [what] across [categories] [in YEAR / over time].
Overview:  Overall, [category] had the highest [figure], while [category] had
           the lowest. [Second feature].
Body 1:    [Group/compare the larger bars with selected figures].
Body 2:    [Group/compare the remaining bars with selected figures].
```

## 3. Pie chart(s)

Proportions. Often 2+ pies to compare (two years or two groups). Overview = largest/smallest segment; biggest change if comparing pies.

```
Intro:     The pie charts compare the proportion of [X] in [YEAR1] and [YEAR2].
Overview:  Overall, [category] made up the largest share in both years,
           whereas [category] accounted for the smallest. The most notable
           change was [category rising/falling].
Body 1:    In [YEAR1], [category] accounted for [%], the largest proportion,
           followed by [category] at [%]...
Body 2:    By [YEAR2], the share of [category] had risen/fallen to [%]...
```

## 4. Table

Usually the most data-dense — **selection is everything**. Treat like a bar/line depending on whether there's a time axis. Overview = highest/lowest across the whole table.

```
Intro:     The table gives information about [what] for [categories] [over time].
Overview:  Overall, [highest figure / row], while [lowest]. [Second feature].
Body 1:    [Group the most significant rows/columns + selected figures + compare].
Body 2:    [The remaining significant data + compare].
```

## 5. Process diagram

Present simple **PASSIVE**; sequence markers. **No figures, no comparison** — just describe the sequence. Two kinds: *manufacturing* (linear) or *natural cycle* (cyclical). Overview = number of stages + linear/cyclical + start point + end point.

```
Intro:     The diagram illustrates the process by which [product] is made.
           / the diagram shows the [natural] cycle of [X].
Overview:  Overall, the process consists of [N] main stages, beginning with
           [start] and ending with [end]. (It is a linear / cyclical process.)
Body 1:    At the first stage, [raw material] is [collected]. It is then
           [crushed], before being [heated].
Body 2:    Following this, [the mixture is poured...]. Finally, [the product
           is packaged and distributed].  (Cycle: ...after which the process
           begins again.)
```

## 6. Map (changes over time, or a proposed plan)

Tense depends on the dates (past→past = past passive; present→future plan = present + future). Overview = the overall transformation. **No trend verbs.**

```
Intro:     The maps illustrate how [place] changed between [YEAR1] and [YEAR2].
           / the proposed changes to [place].
Overview:  Overall, the area was transformed from [a largely rural area] into
           [a more developed, built-up one], the most striking change being [X].
Body 1:    In [YEAR1], there was [feature] in the [location]. By [YEAR2], this
           had been [converted into / replaced by / demolished].
Body 2:    In addition, to the [north/south], [a new X was constructed], while
           [Y remained unchanged].
```

## 7. Mixed / multiple charts (e.g. a pie + a table)

Cover **both** visuals; don't run out of time. Overview = one big feature from **each** visual. Body 1 = visual A, Body 2 = visual B.

```
Intro:     The [chart] and the [table] give information about [X].
Overview:  Overall, [one major feature from visual 1], and [one major feature
           from visual 2].
Body 1:    Regarding the [first visual], [grouped data + figures + comparison].
Body 2:    Turning to the [second visual], [grouped data + figures + comparison].
```

# Part 3 — The language bank (the "iterate all" part)

## Paraphrasing the intro (replace the given verb)

show → **illustrate, depict, present, compare, give information about, provide a breakdown of, outline, demonstrate**. Also reword: *the graph / chart / table / diagram / map* + verb; restate the time frame and the categories.

## Overview signposts

`Overall,` · `It is clear that...` · `In general,` · `As can be seen,` · `A striking / notable feature is that...` · `Broadly speaking,`

## Trend verbs — use the verb AND noun form (this doubles your range)

| Direction | Verbs | Nouns |
|---|---|---|
| **Up** | rise, increase, grow, climb, go up, jump, surge, soar, rocket, leap, escalate | a rise, an increase, growth, a climb, a jump, a surge, an upward trend |
| **Down** | fall, decline, decrease, drop, dip, plunge, plummet, sink, go down, dwindle | a fall, a decline, a decrease, a drop, a dip, a downward trend |
| **Stable** | remain stable / steady / constant / unchanged, level off, plateau, stabilise, hold steady | stability, a plateau |
| **Variable** | fluctuate, vary, oscillate | a fluctuation, variation |
| **Peaks/lows** | peak at, reach a peak / high of, hit a low of, bottom out, reach a low | a peak, the highest point, the lowest point, a trough |

**The verb↔noun pattern:** "rose sharply" = "a sharp rise"; "fell gradually" = "a gradual fall." Switching between them is an easy Lexical Resource win.

## Degree — adverbs (with verbs) / adjectives (with nouns)

| Size | Adverb (verb) | Adjective (noun) |
|---|---|---|
| **Small** | slightly, gradually, steadily, marginally, moderately | a slight, a gradual, a steady, a marginal, a moderate |
| **Large** | sharply, dramatically, significantly, considerably, substantially, markedly, steeply, rapidly | a sharp, a dramatic, a significant, a considerable, a substantial, a marked, a steep, a rapid |

e.g. "Sales **rose sharply**" = "there was **a sharp rise** in sales." Speed: *rapidly / swiftly* vs *slowly / gradually*.

## Comparison & contrast

`higher / lower than` · `greater / less than` · `more / fewer than` · `twice as high as` · `three times as many as` · `the highest / lowest` · `the second highest` · `compared with / to` · `whereas` · `while` · `in contrast` · `by comparison` · `comparable to` · `similar to` · `roughly the same as` · `to outnumber` · `to exceed / surpass` · `to double / triple` · `half / a quarter of`

## Proportion / fractions (pie charts, parts of a whole)

`account for` · `make up` · `constitute` · `comprise` · `represent` · `the largest / smallest proportion / share / percentage / segment` · `a majority / minority` · `a quarter (25%)` · `a third (≈33%)` · `half (50%)` · `two-thirds` · `three-quarters` · `the vast majority` · `a small fraction`

## Approximation (don't repeat exact figures every time)

`approximately` · `around` · `about` · `roughly` · `just over` · `just under` · `nearly` · `almost` · `well over` · `in the region of` · `or so` · `slightly more / less than` · (and for precision) `exactly` · `precisely`

## Quantity / figure phrases

`the figure for X stood at / reached / was recorded at...` · `the number of X...` · `a figure of...` · `X rose to a high of...` · `at a rate of...`

## Time references

`between X and Y` · `from X to Y` · `over the period` · `throughout the period` · `during` · `in the first / last decade` · `by the end of` · `at the start / outset` · `in [YEAR]` · `over the following years` · `by [YEAR]` · `in the subsequent years`

## Process sequencers + passive forms

Sequence: `First / Firstly` · `To begin with` · `Initially` · `At the first stage` · `Then` · `Next` · `After that` · `Subsequently` · `Following this` · `Once X is [done]` · `At this point` · `Meanwhile` · `Finally` · `Lastly` · `In the final stage`
Passive patterns: *is collected* · *is then crushed* · *before being heated* · *having been cooled* · *after which it is packaged* · *is transported to...*
Cyclical: "...the process is **cyclical**, after which it begins again."

## Map language

**Location:** `in the north / south / east / west` · `in the north-east` · `on the outskirts` · `in the city centre` · `adjacent to` · `next to` · `opposite` · `beside` · `surrounding` · `to the left / right of` · `along` · `near`
**Change verbs (past passive):** `was built / constructed` · `was demolished / knocked down / pulled down` · `was converted into` · `was replaced by` · `was relocated to` · `was expanded / extended` · `was redeveloped` · `was transformed into` · `was added` · `was modernised` · `remained unchanged` · `trees were cleared` · (active) `the area **underwent** development`
**Future plan:** `will be built` · `is set to be replaced` · `is planned / projected`

# Part 4 — Tense cheat sheet

| Situation | Tense |
|---|---|
| Past, finished dates | Past simple |
| Future / projection ("by 2030") | will / is projected / is expected to |
| Timeless process | Present simple (passive) |
| Pie/table proportions with no date | Present simple |
| Map: past → past | Past / past-perfect passive |
| Map: present → future plan | Present + future |

# Part 5 — Common mistakes (these cap your score)

- **No overview / vague overview** (→ Band 4–5) — always write one.
- **Specific numbers in the overview** — keep figures out of it.
- **Data-dumping** every figure instead of selecting + grouping.
- **Describing without comparing** (for data charts).
- **Trend verbs on a map or process** — wrong tool.
- **Adding opinion / a conclusion** — Task 1 has neither.
- **Wrong/inconsistent tense.**
- **Under 150 words** (penalised) or **over ~200** (steals Task 2 time).
- High-frequency **article / plural / preposition** errors around figures ("a 20% increase", "the number of people", "an increase **in** X").
